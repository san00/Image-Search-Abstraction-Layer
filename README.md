Image Search Abstraction Layer
=========================

```Search for images and return info from the API in json format```

**Example call**

https://abstraktion.glitch.me/api/search/ ```+ your search term here```

**Example JSON response**
	
url	"https://lumiere-a.akamaihd.net/v1/images/gallery_bears_thumbnail_1a557998.jpeg?region=0,0,450,338"
snippet	"Bears | Disneynature"
thumbnail	"https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcToNPNOLDBf-hQd1qi2egjaEbGOU-slJIGpABzAYxbLqLQIz-hHDyAVHR0"
context	"http://nature.disney.com/bears"

**To view the API search history use**

https://abstraktion.glitch.me/api/history

**Live preview** 

[View it here](https://abstraktion.glitch.me/api/search/bears)


**Built with**

Nodejs, mongodb, express, mongoose, mlab, JavaScript, HTML, CSS.
